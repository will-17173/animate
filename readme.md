#animate


 > 以简单的data-animate属性和纯JS方式实现CSS3动画

##how to demo

1. checkout
1. run `npm install`
1. run `spm install`
1. run `grunt`
1. view files in `/demo`

##how to use

1. run `spm install pandora/animate`
1. write `require('pandora/mvote/VERSION.NUMBER/animate')`

##find examples

1. view the source files in '/src'
